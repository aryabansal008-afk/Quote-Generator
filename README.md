# Quote-Generator
Built a Random Quote Generator using JavaScript that displays inspiring quotes along with the author’s image.

A simple Random Quote Generator built using HTML, CSS, and JavaScript.
The application displays a random inspirational quote along with the author's name and image whenever the user clicks the button.
## Features
* Displays a random quote from a predefined list
* Shows the author name and corresponding image
* Updates content dynamically using JavaScript DOM manipulation
* Simple and clean user interface
## Technologies Used
* HTML – Structure of the application
* CSS – Styling and layout
* JavaScript – Logic for random quote generation and DOM updates
## How It Works
A collection of quotes is stored in a JavaScript array of objects.
Each object contains: quote, author and image.
When the button is clicked:A random index is generated.
The quote, author name and image are updated dynamically on the page.
